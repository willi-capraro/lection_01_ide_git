# Mitschrift

## Lektion 01: IDE + GIT

### Shell Befehle

- `cd` Navigieren in Pfaden 
  - `../` übergeordneter Ordner
  - `.` 
- `ls` Ordner und Datein auflisten 
- `mkdir` Ordner erstellen
- `touch` Datei erstellen (nicht für Powershell)
- `rm` Detei/Ordner löschen (benötigt manchmal weitere Argumente)
